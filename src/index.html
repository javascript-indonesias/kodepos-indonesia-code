“Aplikasi Kode Pos Indonesia”

Versi 1.0.0 - GulajavaMinistudio @ 2020

Aplikasi untuk mencari kode pos berdasarkan daerah kecamatan dan provinsi di Indonesia. Pengguna cukup memasukkan provinsi, pilihan kota atau kabupaten, dan kata pencarian. Kata kunci pencarian dapat berupa nama kecamatan, nama kelurahan, dan potongan nomor kode pos yang ingin dicari. Sumber data kode pos di dalam aplikasi ini bersumber dari repository Github API data publik di Github https://github.com/farizdotid/DAFTAR-API-LOKAL-INDONESIA
